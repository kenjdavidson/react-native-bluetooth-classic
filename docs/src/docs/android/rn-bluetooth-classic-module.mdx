---
title: RNBluetoothClassicModule
description: "Implements the communication and management of Android Bluetooth"
---

At this point you shouldn't need to modify or use the `RNBluetoothClassicModule` directly, it's accessible through the React Native [RNBluetoothModule](/react-native/rn-bluetooth-classic/).

Once configured it will provide all communication through the configured:

- [ConnectionAcceptor](/android/connection-acceptor/)
- [ConnectionConnector](/android/connection-connector/)
- [DeviceConnection](/android/device-connection)
